What is Python?

Python is like a special language that people use to give instructions to computers. Just like people can use words and sentences to talk to each other, programmers use Python to talk to computers.

When you use Python, you write a set of instructions called code, which tells the computer what to do. These instructions can do all kinds of things, from adding numbers to drawing pictures to playing music.

Python is popular because it's very easy to learn and use. It has a simple syntax, which means that the instructions you write look like regular English sentences and are easy to understand. This makes it a great language for beginners who are just starting to learn how to code!

How can Python be used?

Python has many applications. Some of the best ways to use it are for making bots, automating repetitive tasks, making web-sites, and crunching numbers.

A practical example is this: Imagine you have a folder full of images that you want to resize to a specific size. Manually resizing each image can be time-consuming and very tedious. So, instead, you can write code in Python to automate this task. In other words, you can write code in Python to re-size all the images, so you don't need to manually re-size them yourself!

Basic Syntax

Programming syntax refers to the rules and structure that a programmer must follow when writing code in a programming language. It's similar to the grammar rules in a language like English or Spanish, where words and phrases must be arranged in a specific way to make sense.

Syntax errors can occur when the programmer violates one of these rules, causing the computer to be unable to understand the code. As a result, it's important for programmers to pay close attention to the syntax of their code to ensure that it's correct and can be executed without errors.

Printing

Printing is a basic function in Python that allows you to display text or values to the screen.

print("hello, world!)

In this example above, we're using the print function to display the text "hello, world!" to the screen if we were to run the code (we wil do this later).

Comments

In Python, comments are lines of text that are ignored by the interpreter when the code is executed. Comments are used to provide explanations, notes, or reminders for yourself or other programmers who may read the code later.

In Python, comments start with the '#' symbol and continue to the end of the line. For example:

print("hello, world!) #this is a comment

Comments can be used to explain the purpose of a piece of code, how it works, or what it's intended to do. Furthermore, comments can be used to temporarily disable or "comment out" a piece of code that you don't want to run. This can be helpful for debugging or testing purposes. Lastly, comments can be used to make notes or reminders for yourself or other programmers. For example, you might use a comment to remind yourself to come back and change a certain piece of code later.

You can think of comments as the notes you take in your notebook for class, as you will hopefully come back to them later.
